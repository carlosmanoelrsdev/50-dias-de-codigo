# 50 Dias de Código

Repositório criado para registrar minha jornada no desafio **50 dias de código**, com foco em prática diária, evolução contínua e construção de consistência nos estudos e projetos.

## Objetivo

O objetivo deste repositório é documentar minha evolução ao longo de 50 dias de programação, reunindo exercícios, testes, scripts e pequenos projetos desenvolvidos durante esse período.

## Estrutura do repositório

Este repositório pode conter:

- Exercícios práticos
- Scripts em Python
- Testes de lógica de programação
- Experimentos e estudos
- Pequenos projetos desenvolvidos ao longo do desafio

## Tecnologias utilizadas

De acordo com a composição atual do repositório:

- Python

## Proposta do desafio

Durante 50 dias, a ideia é manter uma rotina de prática em programação, buscando:

- Desenvolver consistência
- Aprimorar lógica de programação
- Explorar novas ideias
- Registrar aprendizado
- Criar o hábito de programar todos os dias

## Organização diária

A proposta é que cada dia represente um avanço, seja com:

- resolução de problemas
- implementação de scripts
- estudo de conceitos
- melhorias em códigos anteriores

## Aprendizados

Este repositório também serve como um histórico da minha evolução, permitindo acompanhar:

- minha constância nos estudos
- os conceitos praticados
- os erros e melhorias ao longo do processo
