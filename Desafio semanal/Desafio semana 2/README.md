## versão Final do Gerenciador de Tarefas (CRUD)

Este programa foi desenvolvido durante o projeto 50 dias de código, utilizando Python.
O Gerenciador de Tarefas é um sistema simples em terminal que permite criar, listar, atualizar, remover e buscar tarefas utilizando um menu interativo.

## Funcionalidades

- Criar novas tarefas com título e status padrão (pendente)

- Listar todas as tarefas cadastradas com ID, título e status

- Atualizar tarefas existentes:

    - Alterar o título

    - Alterar o status (pendente ou concluído)

    - Remover tarefas pelo ID

    - Buscar tarefas específicas pelo ID

    - Menu interativo com validação de entradas

## Estrutura do Projeto

- Modelo de tarefa

    - Representa uma tarefa com id, título e status

- Gerenciador de tarefas

    - Responsável por criar, armazenar, atualizar, remover e buscar tarefas

- Menu interativo

    - Exibe opções ao usuário e captura a escolha

- Arquivo principal (main)

    - Controla o fluxo do sistema e direciona as ações com base no menu
