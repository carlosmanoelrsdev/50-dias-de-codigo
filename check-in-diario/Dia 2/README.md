# Sistema de Alistamento Militar

Este sistema foi desenvolvido para estudo da linguagem **Python** durante o projeto **50 dias de código**.

Ele calcula a situação de alistamento de uma pessoa com base no ano de nascimento.

---

## Funcionalidades

- Calcula a **idade** do usuário com base no ano de nascimento.
- Informa o **ano do alistamento**.
- Indica se o usuário deve se alistar:
  - **Este ano** (idade = 18)
  - **Já passou** (idade > 18) e calcula **multa de R$6 por ano de atraso**
  - **Ainda não chegou** (idade < 18)
