## Tela Inicial do Programa de Gerenciamento de Tarefas

Este programa foi desenvolvido durante o projeto 50 dias de código, utilizando Python.
A tela inicial representa o primeiro contato do usuário com o sistema e é responsável por apresentar o menu de opções disponíveis.

Por meio desse menu, o usuário pode selecionar a funcionalidade desejada, como criar, listar, atualizar, remover ou buscar tarefas.
Após a escolha, o sistema realiza o redirecionamento automático para a função correspondente, garantindo uma navegação simples, organizada e intuitiva.

## Funcionalidades

Atualmente, o sistema oferece as seguintes funcionalidades:

- Criar novas tarefas

- Listar tarefas cadastradas

- Atualizar tarefas existentes

- Remover tarefas

- Buscar tarefas pelo ID

- Encerrar o sistema de forma controlada

## Estrutura do Projeto

O projeto está organizado em módulos, seguindo boas práticas de programação:

menu.py
Responsável por exibir o menu interativo e capturar a opção escolhida pelo usuário.

main.py
Responsável pelo controle principal do sistema, redirecionamento das opções e execução das funções.

Essa separação torna o código mais legível, organizado e fácil de manter.

## Como Executar o Programa

Para que o sistema funcione corretamente, o arquivo que deve ser executado é o main.py, pois ele é o ponto de entrada do programa.
```bash
cd caminho/do/projeto
python main.py
```
