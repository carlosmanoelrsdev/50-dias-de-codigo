## Potenciação em C

Este programa foi desenvolvido para estudo da linguagem C durante o projeto 50 dias de código.
Ele calcula a potência de um número inteiro com base nos valores informados pelo usuário.

## Funcionalidades

- Entrada de dois valores inteiros (base e expoente)

- Cálculo da potência utilizando laço de repetição

- Exibição do resultado no terminal

- Tecnologias utilizadas

- Linguagem C

- Compilador GCC
