## Primeira versão do Zoológico Fácil (Sistema de Controle de Animais)

Este programa foi desenvolvido durante o projeto **50 dias de código**, utilizando **Python**.  
O **Zoológico Fácil** é um sistema simples para cadastrar animais, controlar alimentação e emitir sons, simulando a gestão básica de um zoológico.

## Funcionalidades

- Adicionar novos animais ao zoológico
- Listar todos os animais cadastrados com ID e espécie
- Emitir o som característico de cada animal
- Alimentar os animais cadastrados

## Pontos a se atentar

No momento, o sistema possui apenas a **lógica principal**, sem validação completa de entradas.  
Isso significa que:

- É possível inserir valores inválidos ou vazios
- O sistema não impede duplicação de animais
- Valores incorretos podem gerar comportamento inesperado

O sistema será aprimorado futuramente com:

- Controle de IDs automático mais robusto
- Validação de entradas
- Remoção e edição de animais
- Persistência de dados (salvar e carregar animais)

## Exemplo de uso

Ao executar o programa:

1. Você verá o menu com opções de cadastro, emitir som, alimentar ou listar animais  
2. Ao adicionar um animal, o usuário deve informar:  
   - Espécie  
   - Categoria (Carnívoro, Herbívoro, Onívoro, etc.)  
   - Tamanho (Pequeno, Médio, Grande, Gigante)  
   - Som característico  
3. Cada animal recebe um **ID de registro** sequencial ao listar
