## Sistema de Controle de Despesas (Financeiro Pessoal)

Este sistema foi desenvolvido para estudo da linguagem Python durante o projeto 50 dias de código.
Ele calcula o percentual de gastos mensais e o saldo final com base no salário e nas despesas informadas pelo usuário.

## Funcionalidades

- Calcula o total de gastos mensais
- Calcula o percentual do salário gasto
- Calcula o saldo mensal final
- Informa a situação financeira do mês:
    - Sobrou dinheiro (gastos < 80% do salário)
    - No limite (entre 80% e 100%)
    - Acima do limite (gastos > 100%)
