## Primeira versão do Controle Fácil (Sistema de Controle de Vendas)

Este programa foi desenvolvido durante o projeto 50 dias de código, **linguagem python**.
O **Controle Fácil** é um sistema desenvolvido para lojas, onde ele realizará cálculos simples comuns no meio do empreendedorismo.

## Funcionalidades

- Calcula o número total de itens vendidos
- Calcula o valor total das vendas
- Mostra a maior venda realizada no dia
- Mostra a menor venda realizada no dia

## Pontos a se atentar

No momento, o sistema está somente com a lógica principal, sem tratamento de erro e permitindo qualquer tipo de entrada, incluindo valores negativos, positivos e caracteres inválidos, resultando em erros e quebra do código.
O sistema será aprimorado o mais breve possível.
