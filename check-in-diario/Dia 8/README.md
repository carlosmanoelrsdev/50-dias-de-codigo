## Treino de Classes em Python

Este projeto foi desenvolvido para estudo da linguagem Python durante o projeto **50 dias de código**.
O objetivo principal é praticar Programação Orientada a Objetos (POO) por meio da criação e utilização de classes.

- Neste projeto, foi criada uma classe chamada Livro, onde o sistema informa se um livro está disponível ou não, utilizando valores numéricos:

    - 1 → disponível

    - 0 → não disponível

## Funcionalidades

- Criação de objetos da classe Livro

- Armazenamento das informações:

- Título

- Autor

- Ano de publicação

- Disponibilidade

- Verificação e exibição do status do livro (disponível ou não)

## Tecnologias utilizadas

Python 3.x
